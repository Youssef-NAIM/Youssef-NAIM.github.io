Mapping and Autonomous Navigation with LIMO Robot

In this project, I worked on enabling a LIMO mobile robot to autonomously explore and navigate through an unknown environment. Using ROS, I implemented SLAM with the GMapping package to allow the robot to build a real-time map of its surroundings. I then integrated AMCL for localization and move_base for path planning and obstacle avoidance.This project was a hands-on experience in robotics, combining perception, real-time decision-making, and autonomous movement. Watching the robot build its own map and navigate it on its own was both exciting and rewarding.

Automation of a Fishertechnik Multi-Processing Station Model

This project focused on implementing a complete automation solution for a Multi-Processing Station (MPS) of FisherTechnik, utilizing Siemens Simatic S7-1200 PLC and TIA Portal software for programming and control. The project was completed in two main phases:

1. Design and Build the Automation Solution: We began by analyzing the MPS system's functionality, understanding each process step, and identifying the necessary sensors and actuators. Based on this understanding, we programmed the Simatic S7-1200 PLC to orchestrate the automation sequence with precision.

2. Develop the Human-Machine Interface (HMI):In this phase, we focused on creating an intuitive and user-friendly interface for controlling the MPS model. The HMI included features like a start button, emergency stop, and status indicators. Every detail of the interface was designed to ensure smooth and natural interaction. The HMI was then uploaded to the KTP700 Basic module, thus bringing the system to life.

This project combined both automation programming and user interface design to enhance the operational efficiency and usability of the MPS system.
